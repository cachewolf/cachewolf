Die CacheWolf Dokumentation wird im DocBook Format geschrieben. Wer schon einmal selber eine HTML Seite erstellt hat, wird bei der Arbeit mit DocBook gewisse Parallelen erkennen. Trotzdem kann man sich mit speziellen Editoren, die DocBook direkt unterstützen, das Leben deutlich leichter machen.

Ursprünglich wurde im CacheWolf SVN Repository auch ein DocBook Editor für Windows mitgeführt. Solche Drittsoftware veraltet jedoch recht schnell. Wer weiterhin eine Version von XMLMind benutzen möchte, kann sich eine aktuelle Version für Windows oder Java beim Anbieter herunterladen. Die Software ist free for non comercial use. Als Alternative bietet sich eventuell auch Serna Free an, der unter GPL steht und neben dem Quelltextdownload auch als fertiges Paket für Windows, Mac OS X und Linux zur Verfügung steht.
