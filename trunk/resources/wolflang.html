WolfLanguage Commands:
each command is terminated
with a ";".

Basic Operations
Assignment:
var = value;
Example: a = 2;

Math
+, -, *, /, (, ):
Example: a = 1 / 2 + 34 - 8 * 12;
or a = (1 / b + 34 - b) * 12;

Root, Power
srqt(), ^:
Example: a = sqrt(2) ^ 5;

Trigonometric functions
cos(), sin(), tan(),
atan(), asin(), acos():
Example: a = cos(1);
or a = atan(b);

Text
rot13()

Showing values
show(), "some text" show() "more text":
Example: "N 34 12." show(a) " :the " rot13("gnetrg");
